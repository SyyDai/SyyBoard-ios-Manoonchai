# Manoonchai SyyBoard
Typing with Manoonchai layout using a hardware keyboard on iOS.

## Layouts
1. System
1. Qwerty
1. Colemak Mod-DH
1. Thai Manoonchai (‘’“”… at QWERT and «» at NM)
1. Tai Tham Manoonchai

## Build
- Xcode v. 26.0
